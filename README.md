# Inner Demons RPG

A mobile web game where you capture and battle manifestations of internal struggles.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Configured for Vercel. Push to main branch to deploy.

## Tech Stack

- Phaser 3
- TypeScript
- Vite
- Vercel
